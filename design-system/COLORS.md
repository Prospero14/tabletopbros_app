# Color Palette - TableTop Bros

Полная палитра цветов из Android проекта.

## Primary Colors (Deep Purple)

```xml
<color name="primary">#673AB7</color>
<color name="primary_dark">#512DA8</color>
<color name="primary_light">#9575CD</color>
```

- **Primary**: `#673AB7` - Основной фиолетовый цвет бренда
- **Primary Dark**: `#512DA8` - Тёмный вариант для статус-бара и акцентов
- **Primary Light**: `#9575CD` - Светлый вариант для фонов и подсветки

## Secondary Colors (Juicy Orange)

```xml
<color name="secondary">#FF6D00</color>
<color name="secondary_dark">#E65100</color>
<color name="secondary_light">#FF9E40</color>
```

- **Secondary**: `#FF6D00` - Основной оранжевый цвет
- **Secondary Dark**: `#E65100` - Тёмный вариант
- **Secondary Light**: `#FF9E40` - Светлый вариант

## Accent Colors (Warm Orange)

```xml
<color name="accent">#FF9A5C</color>
<color name="accent_light">#FFB88C</color>
```

- **Accent**: `#FF9A5C` - Акцентный цвет для FAB и важных элементов
- **Accent Light**: `#FFB88C` - Светлый вариант акцента

## Background Colors

```xml
<color name="background">#FAFAFA</color>
<color name="background_dark">#121212</color>
<color name="surface">#FFFFFF</color>
<color name="surface_dark">#1E1E1E</color>
```

<color name="error">#D32F2F</color>

<color name="success">#388E3C</color>

## Chat Colors

```xml
<color name="chat_bubble_own">#80B39DDB</color>
<color name="chat_bubble_other">#E0FFFFFF</color>
<color name="poll_background">#80FF9E40</color>
```

- **Chat Bubble Own**: Полупрозрачный фиолетовый для своих сообщений
- **Chat Bubble Other**: Полупрозрачный белый для чужих сообщений
- **Poll Background**: Полупрозрачный оранжевый для фона опросов
